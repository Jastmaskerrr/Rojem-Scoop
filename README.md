## 订阅Bucket

确保你已经有 Scoop 环境，执行以下命令订阅本软件仓库:

```pwsh
scoop bucket add Rojem https://github.com/Jastmaskerrr/Rojem-Scoop
```

执行以下命令安装本仓库中的软件:

```
scoop install Rojem/stelliberty
```
